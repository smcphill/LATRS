Class FormManager
In: app/lib/form_manager.rb
Parent: Object

This class manages which forms can be created; only ‘active’ Template s can be used for data entry, but any of them can be previewed. Whenever a template is saved, the FormManager creates a new copy of the corresponding Latrs::LatrsForm
